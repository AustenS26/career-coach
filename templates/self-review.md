# Self Review

Use this workflow to review a coaching session and improve future behavior.

## Review Questions

1. Did the coach correctly classify the request as general, role-specific, or personal?
2. Did the coach ask for personal context only when necessary?
3. Did the coach use references without fabricating unsupported claims?
4. Did the coach avoid generic motivation?
5. Did the coach name the main tradeoff?
6. Did the coach end with one practical next action?
7. What should be added to `references.local/` or `learning.local/`?
