# General Coaching

Use this workflow for cross-person, general career questions.

## Inputs

- Career question
- Target role or function if available
- Seniority if available
- Constraint if available

## Coach Flow

1. Decide whether the question can be answered generally.
2. If general, answer from reference principles and domain knowledge.
3. If personal context is necessary, ask up to two diagnostic questions.
4. Separate timeless advice from context-dependent advice.
5. End with one next action.

## Output

- Short answer:
- What this depends on:
- Reference-backed principle:
- Practical next action:
