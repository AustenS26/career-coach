# Lenny Podcast Starter Reference

This is a starter reference schema for using Lenny's Podcast as source material in a career coach.

Do not paste full transcripts or long copyrighted summaries into this public repository. Keep public notes short, source-linked, and transformative. Store detailed personal or licensed notes under `references.local/`, which is ignored by git.

## Suggested Source Metadata

For each episode or newsletter-derived note, capture:

```markdown
## Source
- Title:
- Guest:
- URL:
- Date:
- Topics:

## Distilled Principle
One reusable principle in your own words.

## Career Coaching Use
When this source should influence advice.

## Example Prompts
- A general question this source can help answer.
- A personal coaching question this source can help diagnose.
```

## Useful Theme Buckets

Use theme buckets rather than dumping episode-by-episode summaries into the prompt.

| Theme | Coaching Use |
|---|---|
| Product judgment | Product sense, strategy, prioritization, tradeoffs |
| Growth | Loops, retention, activation, distribution, experimentation |
| Career growth | Promotion, scope, visibility, stakeholder trust |
| Leadership | Communication, team design, decision quality |
| Interviewing | Structured answers, product cases, behavioral stories |
| AI and product work | How product roles change with AI tools |

## Recommended Digest Shape

Keep each source note compact:

- Principle: 1-2 sentences
- When to use: 1 sentence
- Watch out: 1 sentence
- Source URL: required

The coach should cite the source label when possible, but it should not pretend to have read full transcripts unless those notes are present locally.
