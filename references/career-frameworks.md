# Career Frameworks Reference

This public reference file contains generic, reusable career coaching principles. It is safe to commit because it does not include personal data or proprietary source material.

## Core Principles

1. Career advice should be contextual, but not every question requires a full personal profile.
2. Strong career moves usually improve at least one of: scope, learning rate, reputation, network, compensation, or optionality.
3. Promotion and seniority decisions often depend on visible impact, repeated judgment, stakeholder trust, and ability to operate with ambiguity.
4. Interview performance is a communication problem as much as a knowledge problem: strong answers show framing, tradeoffs, evidence, and ownership.
5. Resume quality depends on positioning and proof: the reader should know what role you fit and what outcomes you have produced.
6. Offer decisions should compare expected learning, scope, manager quality, company trajectory, compensation, risk, and lifestyle constraints.

## Generic Question Handling

For general questions, the coach can answer without personal context when the user asks about:

- Interview preparation
- Resume structure
- Career decision frameworks
- Offer evaluation dimensions
- How to build influence
- How to turn work into promotion evidence
- How to choose learning projects

For personal recommendations, the coach should ask for:

- Current role and seniority
- Target role or direction
- Main constraint
- Timeline
- Available alternatives
